Client card asset slots
=======================

This directory reserves banner slots for era-specific client cards.

Expected naming convention:

- `assets/client_cards/classic_era/banner.png`
- `assets/client_cards/burning_crusade/banner.png`
- `assets/client_cards/wrath/banner.png`
- `assets/client_cards/cataclysm/banner.png`
- `assets/client_cards/mists_of_pandaria/banner.png`
- `assets/client_cards/warlords_of_draenor/banner.png`
- `assets/client_cards/legion/banner.png`
- `assets/client_cards/battle_for_azeroth/banner.png`
- `assets/client_cards/shadowlands/banner.png`
- `assets/client_cards/dragonflight/banner.png`
- `assets/client_cards/the_war_within/banner.png`
- `assets/client_cards/generic/banner.png`

If a slot is missing, the UI falls back to the built-in gradient treatment.
